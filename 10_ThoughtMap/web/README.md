# ThoughtMap Web v0.1

Local web prototype for ThoughtMap.

## Features

- Upload `.txt`, `.md`, or `.zip`
- Paste text directly
- Split pasted text by delimiter, headings, or blank blocks
- Semantic embedding
- KMeans clustering
- UMAP thought map
- Thought profile chart
- Semantic search

## Install

```powershell
C:/Users/flyin/AppData/Local/Programs/Python/Python313/python.exe -m pip install -r requirements.txt
```

## Run

```powershell
cd D:\ThoughtMap\web
C:/Users/flyin/AppData/Local/Programs/Python/Python313/python.exe -m streamlit run app.py
```

Then open the displayed local URL in your browser.
