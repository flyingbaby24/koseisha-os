# ThoughtMap

ThoughtMap is an experimental project that visualizes thought patterns from lyrics.

Current Features

- Lyric Embedding
- Similarity Search
- Thought Clustering
- Thought Continent Visualization
- Civilization Profile Analysis

Dataset

- 992 self-written song lyrics

Future Goals

- Multi-author comparison
- Similar thinker discovery
- Web-based search engine
- Automatic profile generation
